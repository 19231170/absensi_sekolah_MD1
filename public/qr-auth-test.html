🔐 QR Authentication System

📱 Sistem Login Staff dengan QR Code + PIN

Sistem autentikasi khusus untuk guru dan administrator menggunakan kombinasi QR Code scan dan PIN 4 digit untuk keamanan berlapis.

✨ Fitur Utama:

  • Two-Step Authentication: QR Code scanning + PIN verification
  • Camera Integration: Real-time QR code scanning menggunakan kamera device
  • Manual Input: Fallback input manual untuk QR code
  • Role-based Dashboard: Dashboard berbeda untuk Admin dan Guru
  • Session Management: Secure session handling dengan auto-logout
  • Responsive Design: Optimized untuk tablet dan mobile devices
